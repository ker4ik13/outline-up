### Документация для клиентской части:

#### Установка зависимостей

```bash
npm install
```

или

```bash
yarn
```

#### Запуск проекта в режиме разработки

```bash
npm run dev
```

или

```bash
yarn dev
```

#### Запуск проекта в режиме production

```bash
npm run start
```

или

```bash
yarn start
```

Архитектура приложения: [Feature Sliced Design](https://feature-sliced.design/ru/).
