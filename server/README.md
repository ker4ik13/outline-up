### Документация для серверной части:

#### Установка зависимостей

```bash
npm install
```

или

```bash
yarn
```

#### Запуск проекта в режиме разработки

```bash
npm run start:dev
```

или

```bash
yarn start:dev
```

#### Запуск проекта в режиме production

```bash
npm run start:prod
```

или

```bash
yarn start:prod
```
